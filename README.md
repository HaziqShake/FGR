# FGR - FitGirl Repack Compatibility Checker

A modern, sleek web application that scans your hardware and compares it with the FitGirl Repack library requirements.

## Features
- **Auto-Hardware Detection:** Detect RAM, GPU, and CPU Cores instantly.
- **Smart Matching:** Performance-based estimates (Perfect, Good, Possible, Unsupported).
- **Advanced Filters:** Multi-tag searching, Adult content toggle (Default ON), and Unsupported toggle.
- **Sleek UI:** Minimalist modern design with FitGirl Green accents.

## Tech Stack
- Next.js
- Vanilla CSS
- Supabase
- Playwright (Scraper)
- GitHub Actions (Daily Sync)
